# VoxelKit

**VoxelKit** is a native Apple framework for building living spatial maps that combine real-world geometry, physical objects, IoT devices, and agent behaviour into a unified representation.

## Overview

A single `.botmap` file is a complete description of a space — sufficient for an autonomous agent to navigate, interact with objects, and understand what is happening.

## Platforms

- macOS 26+
- iOS 26+ (capture pipeline)
